# AyurMatters

A web application for Ayurvedic disease management and information system.

## Project Structure

```
AyurMatters/
│
├── frontend/        # HTML / CSS / JS
│   ├── index.html
│   ├── disease.html
│   ├── add-disease.html
│   ├── search.html
│   ├── css/
│   │   └── style.css
│   └── js/
│       ├── disease.js
│       ├── addDisease.js
│       └── search.js
│
├── backend/         # (Coming soon - Spring Boot)
│   ├── src/
│   ├── pom.xml
│   └── application.properties
│
└── README.md
```

## Features

- Disease information management
- Search functionality
- Add new disease entries
- User-friendly interface

## Frontend

The frontend is built using:
- HTML5
- CSS3
- Vanilla JavaScript

## Backend

Backend implementation (Spring Boot) is in progress and will be added soon.

## Getting Started

### Frontend

1. Open `frontend/index.html` in your browser to run the application locally
2. Or serve the frontend folder using any web server
